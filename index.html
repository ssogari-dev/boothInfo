📁 부스 정보 추가

현재 1인 개발자가 직접 부스 정보를 찾으며 수동으로 추가하고 있습니다. 이로 인해 부스 정보가 누락되거나 늦게 추가될 수 있습니다.

'제작자 정보'의 연락처로 추가 요청을 남겨주시면, 해당 건에 대해 우선적으로 빠르게 반영하도록 하겠습니다.

✏️ 부스 정보 수정

등록된 부스 정보에 오류나 변경사항이 있나요?

'제작자 정보'의 연락처로 수정 요청을 남겨주시면, 확인 후 신속하게 수정하겠습니다.

🗑️ 부스 정보 삭제

본 사이트는 부스의 이미지를 별도의 데이터베이스나 서버에 저장하지 않습니다. 게시하신 트윗이나 부스 홍보 게시물의 이미지 URL을 참조하여 보여드리고 있습니다. 아래는 실제로 저장된 데이터 구조 예시입니다:

{
  "eventName": "쏘가리페스 SUMMER",
  "booths": [
    {
      "boothNumber": "A-08",
      "boothName": "쏘가리네 생선 가게",
      "promoUrl": "https://x.com/ssogari_dev/status/1935352923317535017",
      "orderUrl": "",
      "images": ["https://pbs.twimg.com/media/GtvBOkNaQAEDTKJ?format=jpg"], // 트위터 서버 이미지 URL
      "genres": ["붕괴: 스타레일", "호요버스"]
    }
  ]
}

위와 같이 공개된 게시물의 이미지 URL를 참조하는 방식으로, 이미지를 따로 저장하거나 보관하고 있지 않지만, 이 또한 원치 않으실 경우 삭제를 도와드립니다.

삭제를 원하시는 경우 '제작자 정보'의 연락처로 문의해주시면, 해당 정보를 사이트에서 즉시 삭제하고 향후 수집 대상에서도 제외하도록 하겠습니다.
